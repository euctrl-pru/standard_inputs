Below are presented some key values used in the Standard Inputs and that can serve as a reference for other uses.

For any questions relating to this document, please contact EUROCONTROL using the e-mail aviation.intelligence@eurocontrol.int, naming the Standard Inputs in the subject line.

The inflation levels have been calculated based on the Harmonised Index of Consumer Prices (HICP), as the yearly difference in the HICP. The HICP data was extracted from Eurostat HICP - annual data (average index and rate of change) database (prc_hicp_aind).
